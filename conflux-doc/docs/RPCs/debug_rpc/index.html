RPCs#

cfx_getEpochReceipts#

Get one epoch's all receipts in one RPC call

Parameters#

  1. QUANTITY: Epoch number
params: [
"0x1001"
]

Response#

ARRAY - Receipt array

Example#

Request

curl --location --request POST 'http://localhost:12537' \
--header 'Content-Type: application/json' \
--data-raw '{
"id": 1,
"jsonrpc": "2.0",
"method": "cfx_getEpochReceipts",
"params": ["0x1001"]
}'
